## 🎯 Project Overview

End-to-end testing framework using Behavior-Driven Development (BDD) with Playwright automation and Page Object Model design pattern.

## 🚀 Quick Start

```bash
# Install dependencies with uv (fast!)
uv pip install -r requirements.txt

# Install browsers
playwright install

# Run tests
pytest tests/ -v -s
```
